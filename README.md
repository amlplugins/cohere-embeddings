# @amlplugins/cohere-embeddings

Native `cohere-ai` SDK re-exported under the `@amlplugins` namespace.

> Cohere Embed — multilingual & multimodal embeddings (embed-v4) with configurable dimensions.

## Install

```bash
npm install @amlplugins/cohere-embeddings
```

## Use

```ts
import { /* named exports from cohere-ai */ } from "@amlplugins/cohere-embeddings";
// or
import sdk from "@amlplugins/cohere-embeddings";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
